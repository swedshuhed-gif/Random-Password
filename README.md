# Random Password Generator 

**Автор:** Ерныхов Андрей

## Описание

Графическое приложение для генерации случайных паролей с возможностью настройки параметров, сохранением истории в JSON и удобным интерфейсом на `tkinter`.

### Функциональность

- ✅ Настройка длины пароля (4–32 символа)
- ✅ Выбор типов символов:
  - Цифры (0-9)
  - Буквы (A-Z, a-z)
  - Специальные символы (!@#$%^&* и др.)
- ✅ Генерация пароля случайным образом
- ✅ Копирование в буфер обмена
- ✅ История всех сгенерированных паролей
- ✅ Сохранение истории в JSON-файл
- ✅ Загрузка истории из файла при запуске
- ✅ Очистка истории

## Установка и запуск

### Требования
- Python 3.6 или выше
